1. [Search existing issues](https://github.com/your-username/kobedork-app/issues) to avoid duplicates

## Frequently Asked Questions (FAQ)

### General Questions

**Q: Is KOBEDORK free to use?**
A: Yes, KOBEDORK is completely free and open-source under the MIT License.

**Q: Does KOBEDORK collect any user data?**
A: No, KOBEDORK runs entirely client-side. All data (favorites, history) is stored locally in your browser.

**Q: Can I use KOBEDORK offline?**
A: Once installed and built, you can run KOBEDORK locally, but you'll need internet to execute search queries.

### Technical Questions

**Q: What browsers are supported?**
A: Modern browsers including Chrome 76+, Firefox 68+, Safari 13.1+, and Edge 79+. For v1.1 features, Clipboard API support is required.

**Q: What Node.js version do I need?**
A: Node.js version 16 or higher is required.

**Q: How do I update to the latest version?**
A: Run `npm update` or check for new releases on GitHub.

### Security Questions

**Q: Is it legal to use KOBEDORK?**
A: The tool is legal, but how you use it must comply with local laws. Always have proper authorization before testing systems.

**Q: What should I do if I find a vulnerability?**
A: Please report security vulnerabilities through our security policy. See [SECURITY.md](SECURITY.md).

## Reporting Issues

### Before Submitting

- Verify the issue still exists in the latest version
- Search for similar issues
- Prepare detailed reproduction steps
- Include error messages and screenshots if possible

### Issue Template

When creating an issue, please include:

- **Environment**: Browser, OS, version of KOBEDORK
- **Steps to reproduce**: Detailed steps to recreate the issue
- **Expected behavior**: What you expected to happen
- **Actual behavior**: What actually happened
- **Screenshots**: If applicable
- **Additional context**: Any other relevant information
